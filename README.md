# 🏦 Banking Management System (Python + MySQL)

🚀 Overview  
This is a console-based Banking Management System developed using Python and MySQL. The project simulates real-world banking operations such as account creation, deposits, withdrawals, transfers, and bank statement generation.

---

✨ Features  

👤 User Module  
- Create new bank account  
- Deposit money  
- Withdraw money  
- Check account balance  
- Transfer money between accounts  
- View transaction history  
- Generate bank statement (PDF)  

👨‍💼 Admin Module  
- Admin login  
- View all customer accounts  
- View all transactions  
- Manage bank records  

---

🧠 Key Concepts Used  
  
- MySQL Database  
- mysql-connector-python  
- SQL Queries (CRUD operations)  
- ReportLab (PDF generation)  
- Menu-driven console application  

---

🗄️ Database Tables  

- ACCOUNT → Stores user account details  
- TRANSACTIONS → Stores current balance updates  
- TRANSACTION_HISTORY → Stores all transaction records with date/time  

---

⚙️ How It Works  

1. User opens application  
2. Selects option from menu  
3. System performs operation (Create / Deposit / Withdraw / Transfer)  
4. Data is stored in MySQL database  
5. User can generate bank statement in PDF format
